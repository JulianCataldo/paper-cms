# Paper CMS

A document-based headless CMS with API + back office

## Quick start

```sh
# Install all dependencies
npm run deps:i

# Run and watch API + back office
npm run dev

# Open back office
open http://localhost:7777

# Default credentials
# > User : `admin`
# > Password : `password`
```

> ⚠️ **NOT FOR PRODUCTION** ⚠️
